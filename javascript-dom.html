The differences between accessing data from arrays and objects

Arrays are sets of data between [square, brackets] separated by commas. This is a great way to store collections and lists that are accessed by their position within that list. To access the array we use zero-based indexing, so the first item in an array is 0, the second item is 1, the third is 2 etc.

Objects are sets of data between { curly brackets }. The data contained with an object represents a ‘thing’ like a car, person etc. These things then have properties, fields and methods. If the object was a person that data would be name, age, skin colour, height etc. The methods could be what the person can do: Good at coding

While arrays are accessed data according to their position, an object does so by referencing one of the values.
