What is the difference between Margin, Border, and Padding


Explanation of the different parts:

  1. Content - The content of the box, where text and images appear.
  2. Padding - Clears an area around the content. The padding is transparent
  3. Border - A border that goes around the padding and content
  4. Margin - Clears an area outside the border. The margin is transparent


The follow CSS & html code and CSS was used to create the below.


Padding creates transparent space around the content. This can be used on all four sides or ant side one chooses.

  1. Padding: 12px; All four sides to have same padding.>
  2. Padding: 12px 24px 48px 96px; Each value applies to the content in a clockwise direction top, right, bottom and the left.>
  3. Padding: 12px 28px; Here the 12px value applies to the top and bottom and 28px to the right and left>


The border is exactly that, a border

The border goes around the padding and content. In my example i have used "solid". There is other styles which can be used like borders with dashes and dots


Margins

While padding created the transparent space around the content, margin creates this outside the border. Margin makes for ease of reading as well as creating a nicer aesthetic
